# Simulated Payload

Dossier de payloads de test pour CAISSE.

## Structure
- `backend/occupancy/` : payload radar flux magasin
- `backend/sales/` : payload creation vente
- `backend/sync/` : payload sync offline -> serveur
- `backend/jackpot/` : payload configuration jackpot
- `bdd/seed/` : exemples de seed data
- `bdd/mock-data/` : jeux de donnees mock

## Objectif
Centraliser des exemples JSON reproductibles pour QA, debug et demos.
